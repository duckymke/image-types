Image Types: Sea Creatures edition

JPEG

  • JPEG stands for Joint Photographic Experts Group
  • 30% of images on the web are JPG format
  • JPEG format requires more processing processing power to show an image

PNG

  • PNG stands for Portable Network Graphics
  • An image type that can provide a transparent background
  • This image type can be used to design logos

GIF

  • GIF stands for Graphics Interchange Format
  • This image type was created by a computer scientist at CompuServe in 1987
  • A GIF image can contain a maxium of 256 colors
